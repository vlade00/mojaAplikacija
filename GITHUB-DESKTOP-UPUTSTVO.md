# GitHub Desktop - Kratko Uputstvo

## Šta je GitHub Desktop?

**Grafički program** za Git (umesto terminal komandi).

- ✅ Lako za početnike
- ✅ Vidiš sve vizuelno (dugmad, promene)
- ✅ Ne moraš da učiš komande

---

## Kako instalirati?

1. **Skini:** https://desktop.github.com/
2. **Instaliraj** (Next, Next, Finish)
3. **Prijavi se** (ili napravi GitHub nalog)

---

## Kako povezati projekat?

### Opcija 1: Novi repozitorijum
1. File → New Repository
2. Name: `hairstudio` (ili `gymApp`)
3. Local Path: `C:\Users\Vlade\Desktop\gymApp`
4. ✅ "Initialize this repository with a README" (opcionalno)
5. Create Repository

### Opcija 2: Povezati postojeći folder
1. File → Add Local Repository
2. Choose: `C:\Users\Vlade\Desktop\gymApp`
3. Add Repository

---

## Kako commit-ovati promene?

1. **Vidiš promene** (levo - fajlovi koji su promenjeni)
2. **Unesi poruku** (dole - npr. "Initial commit")
3. **Klikni "Commit to main"**
4. **Publish repository** (desno gore) - šalje na GitHub

---

## Šta znače dugmad?

- **Fetch origin** - preuzima promene sa GitHub-a
- **Push origin** - šalje promene na GitHub
- **Pull origin** - preuzima i merge-uje promene
- **Commit** - čuva promene lokalno
- **Publish** - prvi put šalje na GitHub

---

## TL;DR

1. Instaliraj GitHub Desktop
2. File → Add Local Repository
3. Izaberi folder projekta
4. Commit → Publish

**Mnogo lakše nego terminal!** ✅
